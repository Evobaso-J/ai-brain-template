1. Read `{{REPO}}/SKILL.md` — routes to the right module based on the task
2. Read the relevant module file (L2) for domain-specific instructions
3. Load `.jsonl` data files (L3) only when the task requires specific entries

**Do not load all files at once.** Max 2 hops to any piece of information.
